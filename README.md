KnoxNights is the ultimate nightlife companion for Knoxville, TN. It connects patrons with real-time drink specials and empowers bar owners with AI-driven marketing tools.

## Features

### For Patrons
*   **Bar Directory**: Discover local bars, view "Vibes" (Chill, Packed, Live Music), and see ratings.
*   **Real-time Deals**: Find active happy hours and drink specials happening right now.
*   **AI Concierge**: Chat with our "Nightlife Concierge" (powered by Gemini) to plan a custom itinerary based on your mood.
*   **Wallet**: Store targeted coupons sent exclusively to you based on your preferences.

### For Bar Owners
*   **Campaign Manager**: Use Generative AI to brainstorm and launch marketing campaigns in seconds.
*   **Targeting**: Send coupons to specific demographics (Students, Beer Lovers, etc.).
*   **Analytics**: Track deal performance and redemption rates.

## Tech Stack
*   **Frontend**: React 19, TypeScript, Tailwind CSS
*   **AI**: Google Gemini API (`@google/genai`)
*   **Icons**: Lucide React
*   **Router**: React Router DOM

## Getting Started

1.  Clone the repository.
2.  Install dependencies.
3.  Set your `API_KEY` in the environment variables for the Gemini API to work.
4.  Run the application.
